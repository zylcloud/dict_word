### 在线字典查询

​	

- 程序功能

  ​	客户度启动进入一级界面（注册,登录,退出)
  ​    客户端登录进入二级界面(查单词,查看历史记录,注销)

  ​    用户可以登录和注册
  ​    通过数据库查询用户账号密码
  ​    通过数据库存储用户账号密码
  ​    用户名不能重复
     多进程技术处理多个客户端请求

  

- 客户端入口

  ```
  dict_client.py
  ```

- 服务端入口

  ```
  dict_server.py
  ```
